Книга наряд-заданий

1. Описание
Электронная книга наряд-заданий предприятия ведет учет всех выполненных, текущих и планируемых работ,
а также предоставляет отчетность в различных формах


2. Запуск проекта:
cd homework_08\ordersbook
pip install -r requirements.txt
python manage.py runserver


3. UPD (к домашке 7):
* Список сотрудников реализован через ListView (http://127.0.0.1:8000/employees/)
* Просмотр карточки сотрудника реализован через DetailView (http://127.0.0.1:8000/employees/1)
* Удаление сотрудника реализовано через DeleteView, (переопределен метод для "архивирования" сотрудника при удалении)
* Создание сотрудника реализовано через CreateView (с помощью формы forms.EmployeeCreateForm)
