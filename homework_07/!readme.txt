Книга наряд-заданий

1. Описание
Электронная книга наряд-заданий предприятия ведет учет всех выполненных, текущих и планируемых работ,
а также предоставляет отчетность в различных формах


2. Запуск проекта:
cd homework_07\ordersbook
pip install -r requirements.txt
python manage.py runserver


3. Реализованные сущности: JobTitles(должность), Employee(сотрудник), Order(наряд)
админка		http://127.0.0.1:8000/admin/
сотрудники	http://127.0.0.1:8000/employees/
наряды		http://127.0.0.1:8000/orders/


p.s. Данный проект планирую дорабатывать на выпускную работу
